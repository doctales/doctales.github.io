org.doctales.reveal

The plugin org.doctales.reveal publishes DITA content to a reveal.js based presentation. You can see an example here: lab.hakim.se/reveal-js.

Installation

Install org.doctales.reveal via the dita command:

dita -install https://github.com/doctales/org.doctales.reveal/archive/master.zip
