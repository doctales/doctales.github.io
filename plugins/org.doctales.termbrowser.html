org.doctales.termbrowser

The plugin org.doctales.termbrowser generates a terminology browser from terminology databases created using the <termentry> topic type that is shipped with the org.doctales.terminology plugin.

Installation

Install org.doctales.terminology via the dita command:

dita -install https://github.com/doctales/org.doctales.termbrowser/archive/master.zip

webhelp-termbrowser

The webhelp-termbrowser transformation type generates a termbrowser based on the <oXygen/> XML webhelp transtype. The webhelp-termbrowser uses the CSS libraries Bootstrap and flag-icon-css to improve the appearance.

Parameters

none

Usage

dita -i terminology.ditamap -f webhelp-termbrowser
