org.doctales.pdf-css

The plugin org.doctales.pdf-css publishes DITA to HTML5 and then passes the output to node-html-pdf. This plugins takes a HTML file as a parameter and returns a PDF document.

Important The development has been stopped due to missing resources.

Installation

Install node-html-pdf via npm:

$ npm install -g html-pdf

Install org.doctales.pdf-css via the dita command:

dita -install https://github.com/doctales/org.doctales.pdf-css/archive/master.zip
