Plugins

The Doctales team maintains a few DITA-OT plugins, to make DITA editing easier.

  • org.doctales.pdf-css
    The plugin org.doctales.pdf-css publishes DITA to HTML5 and then passes the output to node-html-pdf. This plugins takes a HTML file as a parameter and returns a PDF document.
  • org.doctales.pdf
    The purpose of the plugin org.doctales.pdf is to make it easier to configure a pdf2 based transformation. Its distinctive feature is to have a huge configuration file that has to be passed as an argument to the plugin. The plugin then generates XSL templates based on the passed configuration which override the templates of the pdf2 plugin. The main goal is to make configuration of pdf2 based outputs easier for the user without developing a single line of XSLT code.
  • org.doctales.reveal
    The plugin org.doctales.reveal publishes DITA content to a reveal.js based presentation. You can see an example here: lab.hakim.se/reveal-js.
  • org.doctales.terminology
    The plugin org.doctales.terminology is made for developing a terminology database with DITA.
