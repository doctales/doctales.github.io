org.doctales.pdf

The purpose of the plugin org.doctales.pdf is to make it easier to configure a pdf2 based transformation. Its distinctive feature is to have a huge configuration file that has to be passed as an argument to the plugin. The plugin then generates XSL templates based on the passed configuration which override the templates of the pdf2 plugin. The main goal is to make configuration of pdf2 based outputs easier for the user without developing a single line of XSLT code.

Important The development has been stopped due to missing resources.

Installation

Install org.doctales.pdf via the dita command:

dita -install https://github.com/doctales/org.doctales.pdf/archive/master.zip
