Open Source DITA Team

  • We develop open source solutions for DITA teams.
  • Code hosted on github.com/doctales
  • Plugins:
    • org.doctales.terminology - DITA-OT plugin for terminology management
    • org.doctales.reveal - DITA-OT plugin for generating reveal.js based web presentations from DITA files
    • org.doctales.javaProperties2Dita - DITA-OT plugin for converting Java property files to DITA

How to Contribute?

  • Join us: Code, test, write, translate and design.
  • Share ideas and create visions.
  • Give feedback and write bug reports.
